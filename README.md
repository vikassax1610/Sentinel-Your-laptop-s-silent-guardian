# 📖 Overview

Sentinel is a **desktop anti-theft and monitoring system** that helps you keep track of your laptop in real time.

Instead of relying on expensive third-party services, Sentinel provides your own secure monitoring platform where your laptop silently communicates with your backend and dashboard.

The project is being built as a **production-grade portfolio project** following real software engineering practices.
